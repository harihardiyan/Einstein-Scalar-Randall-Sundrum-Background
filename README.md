This repository contains a monolithic, self-contained **JAX implementation** of a 5-dimensional Einstein–scalar background in the Randall–Sundrum (RS) geometry. The solver uses the superpotential method to obtain a consistent first-order Einstein–scalar system, compute curvature invariants, evaluate hierarchy generation, and extract the tensor KK spectrum.

The entire pipeline is implemented in a single Python file for clarity, reproducibility, and scientific auditability.

---

## 1. Physical Motivation

The Randall–Sundrum model uses a warped extra dimension to generate large hierarchies of scales. This project studies a dynamical version of the RS background where the warp factor $A(y)$ is generated by a scalar field $\phi(y)$ through a superpotential $W(\phi)$.

**The goals of this implementation are:**
*   Solve the Einstein–scalar system using a first-order superpotential method.
*   Verify consistency using constraint diagnostics.
*   Compute curvature invariants (Ricci scalar).
*   Evaluate the hierarchy and redshift between UV and IR.
*   Compute the tensor KK spectrum using a Schrödinger operator.
*   Provide a reproducible, audit-grade reference implementation.

## 2. Mathematical Formulation

### 2.1 Metric Ansatz
The 5D metric is defined as:
$$ds^2 = e^{-2 A(y)} \eta_{\mu\nu} dx^\mu dx^\nu + dy^2$$
where $A(y)$ is the warp factor and $y$ is the extra dimension.

### 2.2 Superpotential System
The scalar potential is derived from a superpotential $W(\phi)$. The Einstein–scalar equations reduce to the first-order system:
1. $\phi' = \frac{dW}{d\phi}$
2. $A' = \frac{\kappa_5^2}{3} W(\phi)$

For the chosen superpotential $W(\phi) = W_0 + c_2 \phi^2$:
*   $\phi' = 2 c_2 \phi$
*   $A' = \frac{\kappa_5^2}{3} (W_0 + c_2 \phi^2)$

### 2.3 Analytic Reference Solution
For this setup, the analytic solution used for verification is:
*   $\phi(y) = v_{UV} e^{2 c_2 y}$
*   $A(y) = k y + \frac{v_{UV}^2}{12} (e^{4 c_2 y} - 1)$

### 2.4 Curvature and Constraints
*   **Ricci Scalar ($R$):** $R = -20(A')^2 - 8A''$ (computed via finite differences).
*   **Null Energy Condition (NEC):** $\rho + p_y = (\phi')^2 \geq 0$.
*   **Hierarchy:** Redshift is evaluated as $e^{-A_{IR}}$.

### 2.5 KK Tensor Spectrum
Tensor perturbations satisfy a Schrödinger-like equation:
$$[- \frac{d^2}{dz^2} + V_T(z)] \psi = m^2 \psi$$
where $dz = e^{A(y)} dy$ and the potential is $V_T(z) = \frac{3}{2} A''(z) + \frac{9}{4} (A'(z))^2$.

## 3. Numerical Method

### 3.1 Grid & Integrator
*   **Stretched Grid:** $y \in [0, \pi r_c]$ with adjustable stretching to resolve the IR region.
*   **JAX Integrator:** Fully JAX-compatible adaptive RK45 integrator with step size control and JIT compilation for high performance.

### 3.2 Diagnostics
The solver monitors:
*   $C_1 = \phi' - \frac{dW}{d\phi}$
*   $C_2 = A' - \frac{\kappa_5^2}{3} W$
*   Analytic error vs numerical solution.

## 4. Running the Code

### Requirements
```bash
pip install jax jaxlib numpy matplotlib
```

### Usage
Default run (Solve and Plot):
```bash
python rs_monolith.py
```

Advanced options:
```bash
# Change grid resolution
python rs_monolith.py --Ny 6001

# Compute Kaluza-Klein spectrum
python rs_monolith.py --kk

# Export results to NPZ format
python rs_monolith.py --export

# Run convergence and sensitivity studies
python rs_monolith.py --study
```
